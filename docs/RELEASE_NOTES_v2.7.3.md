# peterhack v2.7.3

## New since v2.7.2

### Camouflage
- Packed PBR local import fixes neon/emissive paint (AMRE channel 7, 31-byte wire)
- Auto material detects emissive from target paint; manual sliders are fallbacks only
- New defaults: both brushes, auto material, all Paint regions, hotkeys on

### Other
- Removed `[ESP]` console spam
- SDK + bridge refresh for Meccha Chameleon 2.9.0

## Install

1. Extract `peterhack-v2.7.3-win64.zip`.
2. Run `peterhack-loader.exe --local --wait`.
3. Update **both** `peterhack.dll` and `bridge/peterhack-bridge.dll` together.
