# peterhack v2.6.3

## Summary
- Menu open/close: **INSERT** or **F10**
- Camo start hotkey: **F1** (preview F2, unpreview F3, stop F4)
- Fixed camo hotkeys missing quick key taps
- Shorter hotkey arm delay after spawn (0.5s)

## Install
1. Extract `peterhack-v2.6.3-win64.zip`
2. Run `peterhack-loader.exe --local --wait`
3. Launch the game (or wait for it)

Enable **Enable camo hotkeys** in the Camouflage tab, close the menu, and use keys in a match.
